# v1.2 — "Your Brain, Your Plan" Design

## Features
1. Weekly Brain Report (Monday notification + in-app card + share card)
2. Milestone Share Triggers (Brain Score thresholds + brain age below real age)
3. Word Scramble game
4. Memory Chain game

---

## 1. Weekly Brain Report

### Push Notification
- **When:** Monday 9 AM local time (repeating)
- **Text:** "Your weekly Brain Report is ready. Brain Score: X (+/-Y)"
- Replaces existing Sunday 7 PM weekly notification

### In-App Card
- Shows on Home screen top (above brain score card) on Mondays until dismissed
- Content:
  - Brain Score change (e.g., 574 → 620, +46)
  - Brain Age trend (e.g., "Brain Age dropped 2 years")
  - Current streak length
  - Best game of the week (highest score exercise)
  - Games played count
- Dismissal: tap card or X button
- `@AppStorage("lastWeeklyReportDismissed")` stores Monday date string to prevent reappearing

### Share Card (cream style, 360x640)
- "WEEKLY BRAIN REPORT" header
- Week date range (e.g., "Mar 3 - Mar 9")
- Brain Score large + delta arrow (green up / red down)
- Brain Age + trend
- Streak + best game
- "Train your brain free — Memori" footer
- Same cream (F7F5F0) style as existing share cards

### Data
- On Monday, look back 7 days of exercises and brain scores
- Compare latest BrainScoreResult to one from 7+ days ago
- Find exercise with highest normalized score that week
- Count total exercises completed that week

## 2. Milestone Share Triggers

### Brain Score Milestones (500, 600, 700, 800, 900, 1000)
- Full-screen celebration modal (reuse StreakCelebrationView pattern)
- Confetti, big number, milestone-specific message
- Share button with pre-rendered cream share card
- Triggered after brain score save (assessment or workout)
- `UserDefaults("highestBrainScoreMilestone")` stores highest celebrated — each milestone fires once ever, never re-triggers even if score drops and comes back

### Brain Age Below Real Age (first time)
- Shown inline on ScoreRevealView / WorkoutCompleteView
- Extra celebratory text + prominent share button
- `@AppStorage("celebratedBrainAgeBelow")` prevents repeat

### Streak Milestones
- Already exist (7/14/30/60/100) via StreakCelebrationView — no changes needed

## 3. Word Scramble Game

### Gameplay
- Unscramble words against a timer
- 10 rounds, words get longer/harder each round
- Timer pressure creates urgency (starts generous, shrinks per round)
- Score based on speed + accuracy (words solved / total + time bonus)

### Domain
- Memory/Speed hybrid
- Feeds into Brain Score via daily workout

### UI
- Scrambled letters displayed as tappable tiles
- Tap letters in order to spell the word
- Shake/clear to reset current attempt
- Clean, minimal style matching assessment aesthetic

---

## 4. Memory Chain Game

### Gameplay
- Growing sequence of items to remember (Simon-style)
- Starts with 3 items, adds 1 each round
- Items: colored shapes, emoji, or symbols on a grid
- Player watches sequence play, then taps items in order
- Wrong = game over, score = longest chain completed

### Domain
- Memory (pure)
- Classic "how far can you get" escalation

### UI
- Grid of items that light up in sequence
- Player taps to replay the sequence
- Clean visual feedback (highlight on tap)
- No lives — one wrong answer ends it
