Apply targeted polish to `commitmentPage`. Surgical, not a rewrite.

**Step 1 — Remove the typewriter; introduce numbered-row layout.**

Replace the typewriter reveal of bullets with a static numbered-row VStack. Each row:

```swift
private struct CommitmentRow: View {
    let index: Int          // 1, 2, 3, ...
    let verb: String        // accent-colored leading word (e.g., "Train", "Block", "Earn")
    let body: String        // rest of the line
    let isVisible: Bool

    var body: some View {
        HStack(alignment: .firstTextBaseline, spacing: 12) {
            Text(String(format: "%02d", index))
                .font(.system(size: 13, weight: .heavy, design: .monospaced))
                .tracking(1.2)
                .foregroundStyle(OB.fg3)
                .frame(width: 28, alignment: .leading)

            (Text(verb)
                .font(.brand(size: 17, weight: .heavy))
                .foregroundStyle(OB.accent)
            + Text(" ")
            + Text(body)
                .font(.brand(size: 17, weight: .semibold))
                .foregroundStyle(OB.fg))
                .fixedSize(horizontal: false, vertical: true)
                .multilineTextAlignment(.leading)
                .frame(maxWidth: .infinity, alignment: .leading)
        }
        .padding(.vertical, 10)
        .opacity(isVisible ? 1 : 0)
        .offset(y: isVisible ? 0 : 10)
    }
}
```

The "accent on the verb" pattern: split each bullet so the leading verb (`Train`, `Block`, `Earn`, `Win`, etc.) is `OB.accent`-colored and the rest of the sentence is `OB.fg`. This matches Differentiation's receipt-row energy but reads warmer because of the active-voice verb-first phrasing.

**Step 4 — Entrance animation matching Welcome's 1.7s arc:**

```swift
@State private var rowsVisible: [Bool] = []

private func animateCommitmentEntrance() {
    let count = commitmentBullets.count
    rowsVisible = Array(repeating: false, count: count)

    let reduce = UIAccessibility.isReduceMotionEnabled
    let baseDelay = 0.32  // hero rises around here
    let stagger = reduce ? 0.0 : 0.10

    for i in 0..<count {
        DispatchQueue.main.asyncAfter(deadline: .now() + baseDelay + Double(i) * stagger) {
            withAnimation(reduce ? .easeOut(duration: 0.18) : .easeOut(duration: 0.30)) {
                rowsVisible[i] = true
            }
        }
    }
}
```

Headline + eyebrow fade in at t=0.10s (use existing `@State var commitmentHeaderVisible: Bool = false` pattern).
Hero rises at t=0.32s (spring response 0.50, damping 0.78–0.82 — matches Welcome bouncer).
Rows stagger from t=0.32s + 0.10s × index.
CTA / hold-target appears at t = baseDelay + count × stagger + 0.30s.
